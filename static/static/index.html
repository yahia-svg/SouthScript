SouthScript

The most rootin'-tootin' programming language in the West

Introduction

SouthScript is a rootin'-tootin' programming language with a Southern twist. It combines familiar programming concepts with a unique Western flair in its syntax and terminology.

Variables

Variables are declared using the THANG keyword:

THANG x = 5
THANG name = "Billy Bob"
THANG is_cowboy = TRUE

Data Types

Numbers (integers and floats):

THANG age = 42
THANG price = 3.99

Text strings:

THANG greeting = "Howdy partner!"

Boolean values:

THANG is_ready = TRUE
THANG is_done = FALSE

Collections (lists):

THANG numbers = [1, 2, 3]
THANG mixed = [1, "two", FALSE]

Operators

Arithmetic operators:

THANG sum = 5 + 3      # Addition
THANG diff = 5 - 3     # Subtraction
THANG product = 5 * 3  # Multiplication
THANG quotient = 5 / 3 # Division

Comparison operators:

THANG is_equal = 5 == 3      # Equal to (false)
THANG not_equal = 5 != 3     # Not equal to (true)
THANG less_than = 5 < 3      # Less than (false)
THANG greater_than = 5 > 3   # Greater than (true)
THANG less_or_equal = 5 <= 3 # Less than or equal to (false)
THANG greater_or_equal = 5 >= 3 # Greater than or equal to (true)

Logical operators:

THANG and_result = TRUE AN' FALSE  # Logical AND (false)
THANG or_result = TRUE OR FALSE    # Logical OR (true)
THANG not_result = AIN'T TRUE      # Logical NOT (false)

Control Flow

Conditional statements:

RECKON age >= 21 THEN
    HOLLER("You can drink")
MIGHTCOULD age >= 18 THEN
    HOLLER("You can vote")
ELSE
    HOLLER("You're too young")

Loops

For loop:

TROT i = 1 T' 5 THEN
    HOLLER(i)

While loop:

THANG count = 0
WHILES count < 5 THEN
    HOLLER(count)
    count = count + 1

Functions

Function definition:

FIXIN' greet(name) -> HOLLER("Howdy, " + name)

Function call:

greet("Billy Bob")

Anonymous functions:

THANG square = FIXIN'(x) -> x * x
HOLLER(square(5))  # Prints 25

Collections

Creating collections:

THANG numbers = [1, 2, 3]
THANG names = ["Billy", "Bob", "Joe"]

Accessing elements:

THANG first = numbers[0]  # Gets 1

Modifying collections:

SHOVE(numbers, 4)       # Adds 4 to the end
THANG last = YANK(numbers, -1)  # Removes and returns last element
THANG combined = STACKON(numbers, [5, 6])  # Combines two collections

Built-in Functions

HOLLER - Prints output:

HOLLER("Hello world!")

SPEAKUP - Gets input:

THANG name = SPEAKUP()

Collection operations:

SHOVE(numbers, 4)
THANG last = YANK(numbers, -1)
THANG combined = STACKON(list1, list2)

FIREUP - Run another script:

FIREUP("other_file.ss")

Error Handling

SouthScript provides descriptive error messages:

Cattywampus! Runtime Problem: 'x' ain't defined
File example.ss, line 5, in <main>

Complete Example

# Calculate factorial
FIXIN' factorial(n) ->
    RECKON n == 0 THEN
        1
    ELSE
        n * factorial(n - 1)

# Main program
THANG num = SPEAKUP()
THANG result = factorial(num)
HOLLER("The factorial of " + num + " is " + result)
